# QR Code Generator

This is a simple web-based QR Code Generator.

## Features

- Generate QR codes from text input.

## Technologies Used

- HTML
- CSS
- JavaScript

## How to Use

1. Open `index.html` in your web browser.
2. Enter the text you want to convert into a QR code in the input field.
3. The QR code will be generated and displayed on the page.
